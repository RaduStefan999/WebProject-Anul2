1. Introduction

1.1 Purpose

Gardening web tutor is an application that teaches users how to plant and harvest various vegetables and fruits and lets them manage their own garden as well as giving them reminders about taking care of plants. The app encourages progress by showing a leader board on the score's of all application users.

1.2 Design

The application offers courses that are managed by admins. Admins can add, edit or remove courses. The application implements a rich text editor for making courses. Users need to authenticate in order to use the application. Adimns are already seeded in the database and they just need to login. Courses are structured into subcourses in an tree manner. Users can manage their own garden by adding and harvesting plants on the application.

2. Steps Of Usage

2.1 Main Page

The main page is where information about the application is given, there are login and register buttons

and anyone can send feedback

2.2 Login Register

If login or register buttons are clicked a form will pop up

2.3 Courses page

After authenticating the user will be redirected to the a dashboard

with the courses page, in which you can view courses and your progress on them, you can search for courses and you can click on course to start it

2.4 Course page

Each course has a video, a structure of subcourses and a text section.

2.5 Leaderboard page

The leaderboard page where users can see their score compared to the highest score users

2.6 My garden page

My garden shows all the your current plants and each plant type has an options button marked with ...

When you open the options a pop up will appear with all the plants of that plant type. If you harvest or remove a plant you can click the x button to remove it from the application.

You can add another plant to your garden by clicking the + button

After clicking the button a pop up will appear with input fields

Reminders are given by the applications in order to help the user take care of his garden, each reminder can be dismissed with the x button or open to display mor details with the ... button

2.7 Admin manage page

Admins can manage courses on this page, the + button is for adding courses, each course has an edit and delete button

The + and edit buttons will open a popup with a form that contains course input fields amongst which a rich text editor
